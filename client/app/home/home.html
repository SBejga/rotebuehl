rotebuehl

user & token manager

User Registration

Restrictions

User registration is only allowed for emails in domain *.dhbw-stuttgart.de.

Register your account with your lehre mail. You will get a confirmation email with a link to activate your account.
After that, you will be able to login via frontend and REST API.

Security

Your password will be saved encrypted and salted.

REST API

To login via REST API send a POST Request to https://it-bejga1.dhbw-stuttgart.de/rotebuehl/auth/local with following JSON body (of course without the {}) 

POST https://it-bejga1.dhbw-stuttgart.de/rotebuehl/auth/local
{
    "email": "{your_registered_email}",
    "password": "{your_chosen_password}"
}

you will receive a JSON response with your login token (JWT) 

{
    "token": "{TOKEN}"
}

Store that token in a secure manner. I recommend to use a password manager like KeyPass or 1Password.

With that token you can use any other API or webservice which will verify the token with this user management service. Just send the Token base64 encoded as HTTP Authorization Header with Type Bearer.
